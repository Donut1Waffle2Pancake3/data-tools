All Data Tools

TinyDataTool provides free browser-based utilities for working with CSV files, JSON data, and tabular datasets. These tools run entirely in your browser, so your data never leaves your device. You can split large CSV files, merge multiple datasets, remove columns, deduplicate rows, convert CSV to JSON, format JSON, and more.

The tools are designed for developers, analysts, marketers, and anyone who works with spreadsheet or structured data. Each tool focuses on one task and can process data instantly without uploads or account registration.

Use the directory below to quickly find the tool you need.

CSV Tools

Split CSV

Split a large CSV file into smaller files with a fixed number of rows.

 Merge CSV

Combine multiple CSV files into a single CSV.

 CSV Column Remover

Remove one or more columns from a CSV file.

 CSV Column Splitter

Split one CSV into two files by selected columns.

 CSV Column Joiner

Combine multiple CSV columns into one with a custom delimiter.

 CSV Deduplicator

Remove duplicate rows from a CSV file.

 CSV Sorter

Sort CSV rows by one or more columns.

 CSV Row Filter

Filter CSV rows by column values or conditions.

JSON Tools

JSON Validator

Validate JSON and find syntax errors with line and column details.

 JSON Formatter

Format and validate JSON data for readability.

 CSV → JSON

Convert CSV data into structured JSON.

 JSON → CSV

Convert JSON data into CSV format.

 JSON → TSV

Convert JSON data into tab-separated format.

Converters

CSV → TSV

Convert comma-separated data into tab-separated format.

 TSV → CSV

Convert tab-separated data into CSV format.

Text Tools

Remove Duplicate Lines

Remove duplicate lines from text with options like case-sensitivity and preserving order.

 Sort Lines

Sort text lines alphabetically with options like numeric sort and case-sensitivity.

 Word, Line, and Character Counter

Count lines, words, and characters instantly in your browser.

 Find and Replace Text

Find and replace text quickly with options like case-sensitivity and replace all.

 Trim Whitespace

Trim whitespace per line and optionally collapse spaces or remove empty lines.

 URL Encoder / Decoder

Encode text for URLs or decode percent-encoded strings back into readable text.
