    Features

    Supported features and limitations

    Supported:

    • Works entirely in your browser with no uploads
    • Extracts multiple ZIP files in one step
    • Saves all files into a single folder

    Not supported:

    • Password-protected ZIP files
    • Extremely large ZIP sets that exceed browser memory limits
    • Direct folder writing in browsers without File System Access API support
